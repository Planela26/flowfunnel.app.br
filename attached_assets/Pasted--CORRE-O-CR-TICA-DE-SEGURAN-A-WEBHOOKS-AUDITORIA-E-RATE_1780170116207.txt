# ETAPA 1 — HMAC COM RAW BODY (CRÍTICO)

A auditoria identificou que os webhooks validam assinatura utilizando:

```js
JSON.stringify(body)
```

Isso é incorreto.

O sistema deve validar a assinatura utilizando os bytes exatos recebidos na requisição.

## Objetivo

Localizar todas as rotas:

* Stripe
* Hotmart
* Kiwify
* Monetizze
* Perfect Pay
* Eduzz
* Qualquer outro webhook

Substituir o fluxo atual por:

```js
const rawBody = await request.text()
```

Validar HMAC utilizando:

```js
rawBody
```

Somente após validar a assinatura:

```js
JSON.parse(rawBody)
```

## Validar

Confirmar que:

* Assinaturas válidas continuam funcionando
* Assinaturas inválidas são rejeitadas
* Nenhum webhook legítimo é quebrado

Executar testes para cada integração.
